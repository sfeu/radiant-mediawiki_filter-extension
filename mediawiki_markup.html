Mediawiki markup is used most notably by Wikipedia. Not all features of Mediawiki markup are supported (yet), refer to the MediaCloth documentation for details.

To see this: Type this:
bold 
'''bold'''
italics 
''italics''
strikethrough 
<del>strikethrough</del>
xsuperscript 
x<sup>superscript</sup>
asubscript 
a<sub>subscript</sub>
code phrase 
<code>code phrase</code>
link to my page 
link to [/my/page my page]
Works only if mediawiki_filter.enable_workarounds? is set to true.
link to Radiant CMS 
link to [http://radiantcms.org Radiant CMS]
Set mediawiki_filter.add_blank? to false if no new browser window should be opened for remote links.
